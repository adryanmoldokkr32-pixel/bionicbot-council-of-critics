# Smart-Contract Constitution

This skill enables bionicbot to translate the user's core values and business rules into code that cannot be bypassed by any AI agent.

## Instructions
1. Define 'Hard-Constraints' (e.g., Never risk >2% of capital, Never share PII).
2. Draft the logic for a 'Digital Constitution' that governs all system actions.
3. Implement automated audits that check every proposed action against the constitution.
4. Ensure that the 'Human-Override' is the only way to modify core constraints.

## Examples
- "Codify a rule that prevents any AI agent from making a trade if the market sentiment is >90% Fear."
